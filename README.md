# Codestart

这个 gem 目前用来生成 rails engine 的初始化代码  
方便团队进行组件开发

组件开发规范参考：  
http://s.4ye.me/DE1J85

## 使用方法

下载 & 安装

```shell
wget https://github.com/mindpin/codestart/raw/0.3.0/codestart-0.3.0.gem
gem install codestart-0.3.0.gem
```

创建代码

```shell
codestart <project_name>
```
